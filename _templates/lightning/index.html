pyspark-hdfs-notebook

A simple PySpark wordcount app that reads from HDFS

What does it do?

  • Connects to a specified Spark cluster
  • Requires the full URL of a file in HDFS on a remote Apache Hadoop cluster
  • Reads the contents of the file
  • Counts the occurrence of words in the file (identified by spaces)
  • Prints the counts for up to 20 words

Notes on permissions

  • This example does not work with an HDFS that requires credentials
  • The file must be readable by *nbuser* (UID 1011 in the *root* group)
